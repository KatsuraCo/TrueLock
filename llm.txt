TrueLock

TrueLock is an offline-first software application for protected file and message delivery.

Core idea:
- create one encrypted capsule file
- send it through any existing channel
- open it only when the required conditions are met

Supported access rules:
- password
- time window
- geo location
- combined AND/OR logic
- visual key in supported workflows

Platforms:
- Android
- Windows

Public pricing:
- recipients open capsules for free
- Pro for creators has two one-time checkout options:
  - 10 USD via hosted crypto checkout for 1 license / 1 device
  - 20 USD via MoonPay card checkout for 2 licenses / up to 2 devices
- business pilots are custom-scoped
- white-label, SDK, and format licensing are custom-scoped

Business evaluation path:
- focused pilot around one workflow
- one stakeholder group
- short validation period
- direct support discussion via support@truelock.pro

Trust boundary:
- access enforcement is application-layer
- a fully compromised client or runtime can bypass local controls
