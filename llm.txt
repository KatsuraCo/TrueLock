TrueLock

TrueLock is an offline-first software application for protected file and message delivery.

Core idea:
- create one encrypted capsule file
- send it through any existing channel
- open it only when the required conditions are met

Supported access rules:
- password
- time window
- geo location
- combined AND/OR logic
- visual key in supported workflows

Platforms:
- Android
- Windows

Pricing:
- recipients open capsules for free
- Pro unlock for creators is a one-time paid purchase
