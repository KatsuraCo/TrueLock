Secure input

VeilType

An Android keyboard privacy layer that encrypts content before it enters another app.

  • Text encryption before sending.
  • Voice, video, photo, and file capsule flows.
  • Works around existing messengers and apps.
  • Built for privacy without messenger migration.
Controlled access

TrueLock

Encrypted capsules where having the file or link is not the same as permission to open it.

  • Password, time, geo, and validation rules.
  • Local cryptography with online validation when conditions require it.
  • Useful for secure sharing, confidential files, and field workflows.
  • Designed for access control after content leaves the device.
