# VeilType Landing

Static landing page for VeilType.

## Files

- `index.html` main page
- `styles.css` visual system
- `app.js` tiny runtime
- `locales/*.js` active site languages
- `assets/` screenshots
- `downloads/VeilType.apk` current Android build
- `llms.txt` AI-readable root summary
- `index.html.md` markdown mirror for agents
- `trust-model.md` and `launch-package.md` machine-readable product context

## Open locally

You can:

- open `index.html` directly in a browser
- or run `start-local.ps1` for a local static server
