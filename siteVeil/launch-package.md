# VeilType Launch Package

## Product name

VeilType

## Tagline

Encrypt before you send: a private Android keyboard for any chat.

## Launch story

1. Enable VeilType Keyboard.
2. Create or receive one 8-emoji shared key.
3. Open any normal chat app and encrypt locally before sending.

## Positioning

- Android-first
- local-only
- one 8-emoji shared key
- no new messenger
- no server
- no account
- no cloud
- no recovery
- text first, with voice/photo/video capsules for richer messages
